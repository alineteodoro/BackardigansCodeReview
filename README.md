# Code Review - Backyardigans

## 📌 Descrição
Projeto criado com o objetivo de treinar Code Review. Aqui temos uma página pessoal para cada integrante do grupo, onde para cada código um integrante diferente revisava o código do outro.

## 🖼️ Conteúdo
Como página temos uma página principal inspirada na tela de perfis da Netflix, com cada integrante identificado como um membro do Backyardigans. Quando entramos em qualquer perfil, temos acesso à página pessoal de cada um.
